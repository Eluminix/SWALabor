# WahlergebnisseApp

## Dependencies

- Python 3.9+

## Docker Setup

Docker Desktop installieren

Im Terminal in VSC eingeben: 

docker compose build --no-cache   // --no_cache wenn node_modules Schwierigkeiten macht

docker compose up
